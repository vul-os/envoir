# Envoir brand

The Envoir mark: an **envelope whose flap is a mesh "V"** — the two upper corners and the
central crease are network **nodes**, and a faint dashed link hints at the wider peer mesh. It
says *mail* + *decentralized network* in one glyph, and nods to *envoir* = envoy/messenger.

## Files

| File | Use |
|------|-----|
| `logo-mark.svg` | The app-tile mark (blue→purple gradient, 128×128, rounded tile). Favicons, app icons. |
| `logo-mono.svg` | Single-color mark via `currentColor` — for light/dark UI, print, watermarks. |
| `wordmark.svg` | Mark + "Envoir" lockup for headers/navbars. |

## Colors

- Gradient: `#5B9DFF` (blue) → `#7C5CFF` (violet) — the Envoir accent.
- On dark: white mark on the gradient tile; on light: use `logo-mono.svg` with a dark `color`.

## Usage

- Keep clear space ≈ the corner radius around the tile.
- Don't recolor the gradient, stretch, or add effects. Use `logo-mono.svg` when one color is
  needed.
- Favicon: `logo-mark.svg` scales down cleanly to 16px (the node dots stay legible).
